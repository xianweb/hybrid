# 如何使用？

使用时，只需要在span节点上分别增加.mui-icon、.mui-icon-name两个类即可

示例

<span class="mui-icon mui-icon-weixin"></span>

效果

# button（按钮）

mui默认按钮为灰色，另外还提供了

  • 蓝色（blue）
  • 绿色(green)
  • 黄色(yellow)
  • 红色(red)
  • 紫色(purple)

# 普通按钮

<button type="button" class="mui-btn">默认</button>
<button type="button" class="mui-btn mui-btn-primary">蓝色</button>
<button type="button" class="mui-btn mui-btn-success">绿色</button>
<button type="button" class="mui-btn mui-btn-warning">黄色</button>
<button type="button" class="mui-btn mui-btn-danger">红色</button>
<button type="button" class="mui-btn mui-btn-royal">紫色</button>

若希望无底色、有边框的按钮，仅需增加.mui-btn-outlined类即可，代码如下：

示例

<button type="button" class="mui-btn mui-btn-outlined">默认</button>
<button type="button" class="mui-btn mui-btn-primary mui-btn-outlined">蓝色</button>
<button type="button" class="mui-btn mui-btn-success mui-btn-outlined">绿色</button>
<button type="button" class="mui-btn mui-btn-warning mui-btn-outlined">黄色</button>
<button type="button" class="mui-btn mui-btn-danger mui-btn-outlined">红色</button>

# switch(开关)

默认状态

示例

<div class="mui-switch">
  <div class="mui-switch-handle"></div>
</div>

效果

默认选中只需要在.mui-switch节点上增加.mui-active类即可

示例

<div class="mui-switch mui-active">
  <div class="mui-switch-handle"></div>
</div>

效果

若希望隐藏on/off文字提示，变成简洁模式，需要在.mui-switch节点上增加.mui-switch-mini类

示例

<!-- 简洁模式开关关闭状态 -->
<div class="mui-switch mui-switch-mini">
  <div class="mui-switch-handle"></div>
</div>
<!-- 简洁模式开关打开状态 -->
<div class="mui-switch mui-switch-mini mui-active">
  <div class="mui-switch-handle"></div>
</div>

效果

# confirm 确认框

语法

示例

<button id="btn" type="button" class="mui-btn mui-btn-blue">按钮</button>

<script type="text/javascript">
	document.getElementById('btn').onclick = function(){
		mui.confirm('你确定要删除吗？','提示',['好的','放弃'])
	}
</script>

效果

# toast 消息提示框

语法

示例

<button id="btn" type="button" class="mui-btn mui-btn-blue">按钮</button>

<script type="text/javascript">
	document.getElementById('btn').onclick = function(){
		// duration 时长 支持数值 long 长 long(3500ms),short(2000ms)
		mui.toast('登录成功！',{ duration:'long'})
	}
</script>

效果
