# HBuilderX安装

# 软件介绍

HBuilder是DCloud（数字天堂）推出的一款支持HTML5的Web开发IDE。 HBuilder的编写用到了Java、C、Web和Ruby。HBuilder本身主体是由Java编写。

# 软件特点

快，是HBuilder的最大优势，通过完整的语法提示和代码输入法、代码块等，大幅提升HTML、js、css的开发效率。

# 软件下载

官网地址：https://www.dcloud.io/hbuilderx.html

选择 download 下载

这里我们选择App完整版

# 解压即可，免安装

该软件是免费软件，无需注册，下载后解压即可使用

WARNING

特别注意：由于软件解压后，大概有4万个左右文件，所以大家要耐心等待，其次不要解压到桌面上，也不要放在含义中文的目录里面。

# 常用快捷键

以下快捷键以 HBuilderX为预设

alt + Q：显示隐藏项目管理器

ctrl + ↑：往上移动当前行代码

ctrl + ↓：往下移动当前行代码

alt + /：激活代码助手

ctrl + shift + F：自动格式化代码

ctrl + L：往下选中代码

ctrl + /：注释代码

ctrl + D：删除光标选中行

ctrl + F：搜索

ctrl + H：搜索替换

复制上一行： 不选中任何代码，直接在某行按 ctrl+c 然后 ctrl+v

# 代码快写

  1. 在body内输入div.abc按下tab键
<div class="abc"></div>
  1. 在body内输入div#abc按下tab键
<div id="abc"></div>
  1. 在body内输入ul>li{小花}*3按下tab键
<ul>
  <li>小花</li>
  <li>小花</li>
  <li>小花</li>
</ul>

4.在body内输入div>a[href='#']{小花}*3按下tab键

<div>
	<a href="#">小花</a>
	<a href="#">小花</a>
	<a href="#">小花</a>
</div>
  1. 在body内输入select>option{$}*5按下tab键
<select name="" id="">
	<option value="">1</option>
	<option value="">2</option>
	<option value="">3</option>
	<option value="">4</option>
	<option value="">5</option>
</select>

6.在body内输入select>option[value='花香']{$}*5>a[href='#']{小花}按下tab键

<select name="" id="">
	<option value="花香">1<a href="#">小花</a></option>
	<option value="花香">2<a href="#">小花</a></option>
	<option value="花香">3<a href="#">小花</a></option>
	<option value="花香">4<a href="#">小花</a></option>
	<option value="花香">5<a href="#">小花</a></option>
</select>

提示

总结一下： 属性值用[]填充 文本内容用{}填充 递增的变量用$填充

# 夜神模拟器安装

# 软件介绍

夜神安卓模拟器，电脑玩手游的新一代神器，夜神安卓模拟器是一款采用类手机界面视觉设计的PC端桌面软件，采用内核技术（基于Android4.4.2版本内核针在电脑上运行深度开发），具有同类模拟器中领先运行速度和稳定的性能。
